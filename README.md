# NoteNest

A beautiful, fast, full-stack note-taking app built with Next.js 15, Prisma, and Tailwind CSS. Deployed on Vercel.

## Features
- Markdown editor with live preview
- User auth via Google/GitHub
- Private notes per user
- Responsive design

## Quick Start
1. Clone: `git clone https://github.com/umayer16/notenest2.git`
2. Install: `npm install`
3. Env: Copy `.env.example` to `.env.local` and add your keys
4. DB: `npx prisma db push`
5. Dev: `npm run dev`

See full setup in the code!

## Deploy
- Push to GitHub → Import to Vercel → Auto-deploys!

Built with ❤️ using Grok.
